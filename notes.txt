How to start the react native bundler:
Inside the project directory $ npm start

Install prettier:
Enable the prttier extenstion
CMD + , and Enable Format on Save

Creating a new project
npx expo-cli init projectName

Install react navigation for stack navigator
npm install react-navigation

Icons install because of expo-cli install
https://github.com/expo/vector-icons

Axios
npm install axios

Setting up a new app
  npm expo-cli init projectName

  // React navigation fix
  npm install react-navigation
  expo install react-native-gesture-handler react-native-reanimated react-native-screens react-native-safe-area-context @react-native-community/masked-view
  npm install react-navigation-stack @react-native-community/masked-view
  npm start -c
